🔧 Guide Débogage F12

Les DevTools expliqués simplement

💬 L'Onglet Console

🎯 À quoi ça sert ?

  • Voir les messages de votre code (console.log)
  • Détecter les erreurs JavaScript
  • Tester du code rapidement

🧪 Démonstration

  1. Appuyez sur F12
  2. Cliquez sur l'onglet Console
  3. Cliquez sur le bouton ci-dessous
  4. Observez les messages

✏️ Testez dans la Console

console.log("Hello World!")
2 + 3
fetch('https://api.quotable.io/random')

🌐 L'Onglet Network

🎯 À quoi ça sert ?

  • Voir tous les appels API
  • Vérifier les codes de statut
  • Voir le contenu des réponses

🧪 Test API

  1. Ouvrez l'onglet Network
  2. Cliquez sur le bouton ci-dessous
  3. Regardez apparaître "random" dans Network
  4. Cliquez dessus pour voir les détails

🎨 Codes de Statut

200 = Succès ✅

404 = Non trouvé ❌

500 = Erreur serveur 💥

👀 Que regarder

Name : Nom de la requête

Status : Code de réponse

Response : Données reçues

🚨 Comprendre les Erreurs

❌ ReferenceError

Uncaught ReferenceError: maVariable is not defined

Problème : Variable qui n'existe pas

Solution : Vérifiez l'orthographe

🌐 TypeError

TypeError: Cannot read property 'content' of undefined

Problème : Objet undefined

Solution : Vérifiez que l'objet existe

📡 Network Error

Failed to fetch

Problème : Problème de connexion

Solution : Vérifiez l'URL et la connexion
