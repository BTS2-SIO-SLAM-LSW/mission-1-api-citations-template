🎣 L'API Fetch

🚀 fetch() = "Aller chercher"

Méthode moderne pour récupérer des données

const response = await fetch(url);
const data = await response.json();

✨ Avantages

• Syntaxe simple
• Promises natives
• Support moderne

🔄 Processus

• Envoyer requête
• Attendre réponse
• Traiter données

⏳ Async/Await

🎭 Le Problème de l'Asynchrone

Internet n'est pas instantané ! Il faut attendre les réponses.

❌ Sans await

const response = fetch(url);
// Promise pending

😵 Confusion

✅ Avec await

const response = await fetch(url);
// Response object
